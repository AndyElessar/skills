## When to Fetch Docs

Fetch latest docs **before** advising on:
- New integration packages (APIs change between versions)
- CLI commands and flags (new options are added regularly)
- Dashboard features (UI evolves with each release)
- Deployment and publishing workflows
- Breaking changes between major versions
